Wf5
===

Workflow Engine (development tools: C#, BPMN)

工作流引擎 

.NET, OpenSource

功能介绍：

1. 参考BPMN标准的流程定义解析；

2. 递归解析实现的工作流引擎；

  2.1 顺序流转；

  2.2 分支/合并；

  2.3 条件表达式；

  2.4 自身循环；

  2.5 撤销/退回/返签；

  2.6 角色资源；
  
3. 集成嵌入Dapper MicroORMapping 框架；

4. 流程定义BPMN文件的缓存读取；

5. 异常日志记录Wflog；

6. 工作流服务WebApi 接口示例；

7. Test 代码示例。
